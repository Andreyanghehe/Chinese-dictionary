## 汉语字典

+ 使用node服务器和vue做成的汉语字典, 可查询字, 词, 成语, 歇后语

+ 数据来源

 - https://github.com/pwxcoo/chinese-xinhua

+ 打包前的vue项目源码

  + https://github.com/Andreyanghehe/vue

+ 使用方法

  + 终端进入下载后的目录

  + 运行命令

    + `npm install`

    + `node index.js`

  + 打开浏览器输入下面网址
    + http://localhost:3000
